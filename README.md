Данная форма была разработана для того, чтобы аккаунт-менеджеры могли добавлять продукты, которые автоматически сохраняются в базе данных.

Также был создан справочник, предназначенный для возможности дополнительного добавления и управления фильтрами в Power BI, что упрощает аналитику и расширяет возможности отчетности.
